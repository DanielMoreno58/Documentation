## Configure Maven in Mac

1. Go to route source with `cd` in iterm or terminal

2. Do the following command: `source .bash_profile`

Bash Profile config

```bash
export MAVEN_HOME=/Applications/apache-maven-3.8.4
export PATH=$PATH:$MAVEN_HOME/bin
```
